# todo-app

## Project setup
```
npm install -g @vue/cli
vue create todo-app
cd todo-app
npm install sass-loader node-sass --save

npm install -D tailwindcss@npm:@tailwindcss/postcss7-compat postcss@^7 autoprefixer@^9
npx tailwindcss init -p
```
